# QSVT Code Attribution

The code in this directory is a modified copy of
[lbwei1016/QSVT-implementation](https://github.com/lbwei1016/QSVT-implementation).

It is kept as a vendored dependency because `qsvtSolver.py` uses the QSVT linear
solver to experiment with quantum linear-system solves inside the Marchenko
reconstruction.

The `Solvers/` subdirectory also preserves its original credits:

- [QSPPACK](https://github.com/qsppack/QSPPACK)
- [QSVT in Qiskit 2021 Europe Hackathon Winning Project](https://github.com/bartubisgin/QSVTinQiskit-2021-Europe-Hackathon-Winning-Project-/tree/main/qsvt_search/src/QSPSolver)
